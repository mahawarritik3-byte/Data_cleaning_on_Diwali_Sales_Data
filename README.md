🪔 Diwali Sales Data Analysis
📌 Project Overview

This project focuses on Data Cleaning, Data Analysis, and Visualization using the Diwali Sales Dataset.
The main objective of this project is to clean raw sales data and extract meaningful business insights through exploratory data analysis (EDA).

The analysis helps in understanding:

Customer purchasing behavior
Sales trends
Product performance
Demographic insights
Revenue generation patterns
📂 Dataset

Dataset Used: Diwali_Sales_Data.csv

The dataset contains information related to:

Customer demographics
Product categories
Orders & sales
States and occupations
Marital status and gender
⚙️ Operations Performed
🧹 Data Cleaning
Removed unwanted columns
Checked and handled null values
Corrected data types
Renamed columns for better readability
Removed duplicate records
📊 Exploratory Data Analysis (EDA)
Gender-wise analysis
Age group analysis
State-wise sales analysis
Occupation-based purchasing analysis
Product category analysis
Top-selling products identification

📷 Project Insights

Some key insights derived from the dataset:

Married women age group 26-35 are major buyers.
Customers from certain states contribute the highest sales.
IT, Healthcare, and Aviation sectors show high purchasing power.
Food, Clothing, and Electronics are among the top-selling categories.
